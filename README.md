# Pacman AI Project 🎮

A simplified Pacman game implemented in Python as an Artificial Intelligence course project.

## Features
- Pacman autonomous navigation  
- Food collection and ghost avoidance  
- AI-based decision making using:
  - A* Search
  - Breadth-First Search (BFS)

## Agent Model
- Model-Based Reflex Agent  
- Goal-Based Agent  

## PEAS Analysis
- **Performance:** Maximize score, avoid ghosts, complete maze  
- **Environment:** Maze, walls, ghosts, food pellets  
- **Actuators:** Up, Down, Left, Right  
- **Sensors:** Positions, food locations, danger map  

## ODESA Analysis
- Fully Observable  
- Stochastic  
- Sequential  
- Dynamic  
- Multi-Agent  
- Discrete State Space
